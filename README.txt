Allow you to add redirect 404 pages to any page you wish, just easily

== Description ==

Just install Plugin and enjoy feature of Advance Redirect 404 Pages Plugin.

This plugin used for redirect 404 pages.

1). After activating plugin go to settings and click on Advance Redirect 404 Pages.

2). in General Option select atatus and add link to "Redirect all 404 pages to".

3). it's Done. Simple Right!!

== Installation ==
1. Upload the entire `advance-rediect-404-pages` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
